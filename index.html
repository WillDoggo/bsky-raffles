Optional password verification

If you want to verify this raffle drawing as official, you can provide an app password here associated with your BlueSky account.

This is not required, but can be used as an additional step to show raffle entrants that this drawing was done by the actual post author.

You can create an app password here.

This tool does not store this password, and only uses it for verifying the post author. The app password can be deleted on BlueSky after the drawing is made.
